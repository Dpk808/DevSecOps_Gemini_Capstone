# AI-Driven Secure Code Auto-Remediation in DevSecOps

This project integrates AI (Gemini API) into a DevSecOps pipeline
to detect and remediate security vulnerabilities automatically
during GitHub Actions.

## Features
- Static code analysis (Bandit)
- Dependency vulnerability scanning (pip-audit)
- AI-generated fixes using Gemini
- Manual or automatic remediation
- Security report artifacts

## Usage
- Push code OR trigger workflow manually
- Choose whether to apply fixes
- Review generated security report
